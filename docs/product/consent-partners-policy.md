# Privacidad y Datos (Borrador)

**Estado:** Documento interno de producto (Hackathon).

## 1. El objetivo de Feria

Feria es una plataforma para ayudar a la gente a llevar sus finanzas de una forma relajada. Para nosotros, la privacidad es prioritaria y no vendemos datos a terceros.

## 2. Cómo manejamos la información

- **Pedimos lo mínimo**: Solo guardamos los datos que necesitamos para que las gráficas, los logros y el tutor funcionen bien.
- **Transparencia**: Siempre queda claro qué datos compartimos de forma interna para que el asistente de inteligencia artificial pueda responder a tus solicitudes.
- **Sin venta comercial**: No compartimos ni vendemos información de nuestros usuarios a aseguradoras, empresas de marketing o plataformas de terceros.

## 3. Uso de audios y textos

Cuando un usuario sube un audio para registrar un gasto rápido, ese archivo se procesa para sacar el texto y el monto. No usamos estos audios ni los mensajes del chat para entrenar nuestros modelos de inteligencia artificial propios. Solo se procesa para resolver el flujo que el usuario solicitó.

## 4. Alineados al ODS 8

Nuestro enfoque principal está atado al ODS 8 (Trabajo decente y crecimiento económico). Buscamos quitarle a las personas el nivel de estrés que genera la gestión financiera diaria. Les damos herramientas accesibles y gratuitas sin tratarlos como producto.
